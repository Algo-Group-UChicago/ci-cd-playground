# CI/CD Playground

A simple Python project for learning CI/CD, GitHub Actions, and automated testing.

## Setup

```bash
pip install -r requirements.txt
```

## Run Tests

```bash
pytest test_calculator.py -v
```

## Exercises

1. **Fix/Add Tests**: Complete the TODO items in `test_calculator.py`
2. **Modify Workflows**: Edit `.github/workflows/ci.yml` to add new steps
3. **Add Linting**: Add a linting step to the CI workflow
4. **Add Docker**: Create a Dockerfile and add a build step
